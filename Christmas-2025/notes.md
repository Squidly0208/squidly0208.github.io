# Christmas 2025 Notes

## Project Overview
Interactive Christmas gift website where family members can enter their name as a secret code to unlock personalized messages and memories.

## Structure
- **index.html** - Main page with login interface and gift display
- **data.js** - Contains all personalized messages and memories for each family member
- **script.js** - Handles unlock functionality and snow animation
- **style.css** - Styling for the page

## Features
- Secret code entry system (uses person's name)
- Personalized messages for each family member
- List of treasured memories
- Animated snow effect background
- Responsive design

## Technical Details
- Code matching is case-insensitive
- Invalid codes display "Invalid code ❄️" error
- Snow animation uses canvas with 120 snowflakes
- Hidden class used to toggle between login and gift display
- Fallback message for people with no specific memories listed: "So many memories, too many to list ❤️"
